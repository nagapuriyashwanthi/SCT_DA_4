# SCT_DA_4 - Business Insights Report (EDA)

## 📌 Task Overview
Performed Exploratory Data Analysis (EDA) 
on a Marketing Campaign dataset as part of 
my Data Analyst Internship at 
SkillCraft Technology.

## 🛠️ Tools Used
- Python
- Pandas Library
- Matplotlib
- Seaborn
- Google Colab / VS Code

## 📊 Dataset
- Marketing Campaign Dataset
- 2,240 rows and 29 columns

## ✅ What I Did
- Loaded and cleaned marketing dataset
- Fixed missing values in Income column
- Converted date column to datetime format
- Created 5 visualizations:
  - Chart 1: Marketing Channel Performance
  - Chart 2: Total Spending by Product Category
  - Chart 3: Campaign Acceptance Rate
  - Chart 4: Income vs Total Spending
  - Chart 5: Education vs Spending
- Generated Business Insights Report

## 📈 Key Insights
- Store channel has highest purchases
- Wines and Meat are top spending categories
- PhD customers spend the most
- Average customer income is $52,247
- Average customer spending is $605

## 💡 Recommendations
- Increase budget for Store channel
- Increase budget for Web channel
- Reduce budget for Deals channel
- Target PhD and Master degree customers
- Focus on Wines and Meat products
